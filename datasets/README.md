# Datasets Folder

This folder contains sample raw datasets used for the Data Warehouse project.

## Structure

### crm/
Customer Relationship Management source data
- customers
- products
- sales

### erp/
Enterprise Resource Planning source data
- categories
- inventory
- transactions

### placeholders/
Empty files used to maintain folder structure in Git

## Notes
- These are sample datasets (30–40 rows each)
- Used only for learning and demonstration
- Not production or confidential data
- Bronze layer source files
